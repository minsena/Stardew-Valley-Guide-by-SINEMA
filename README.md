# Stardew Valley Beginner Guidebook

A cozy, retro, Stardew Valley-themed guidebook for new farmers. Built completely using semantic HTML5 and vanilla CSS. No JavaScript, Tailwind CSS, or external styling frameworks are used.

## Directory Structure

```
stardew-guide/
├── index.html
├── style.css
├── assets/
│   ├── images/
│   ├── icons/
│   ├── cursor/
│   └── backgrounds/
└── README.md
```

## Styling Features
- **Color Theme**: Rich earthy colors (Forest Green, Grass Green, Golden Wheat, Wood, Dirt, Cream, and Sky).
- **Typography**: Retro pixel headings using `'Press Start 2P'` and clean Korean readable sans-serif `'Pretendard'` / `'Outfit'` for body copy.
- **Custom Cursor**: Retro sword and pointer SVG icons loaded as URL data URIs directly inside CSS.
- **Interactive UI**: pure CSS accordions for FAQ using `<input type="checkbox">` toggles, hover animations for cards/buttons, and custom checklist styles.
